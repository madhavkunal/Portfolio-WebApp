Hi, I'm Madhav, a passionate Python developer with a strong interest in the fields of AI and cybersecurity. I love exploring the vast possibilities that Python offers and leveraging its power to build robust and efficient applications.

Education

My Python programming journey started with a comprehensive Udemy online course, The Complete Python Developer, by Andrei Neagoie and the highly acclaimed book; Python Crash Course (Third Edition) by Eric Matthes. The Complete Python Developer course provided me with proficiency in a wide range of Python topics, from the basics and programming fundamentals to advanced topics such as Object-Oriented Programming, Functional Programming, and Data Structures. I also delved into practical applications like web development, machine learning, data science, and automation with popular tools such as Selenium, Kaggle and Jupyter Notebook. The course emphasized hands-on learning, enabling me to build several projects, including this portfolio website you are currently on as well as a web scraper for Hacker News. The Python Crash Course laid a strong foundation in programming concepts such as data structures, functions, modules and introduced advanced tools like pytest for testing and the pathlib module for file handling. This book was instrumental in teaching me how to write clean, efficient code and troubleshoot common programming issues effectively.

My cybersecurity knowledge was really gained through completing my hands-on Google Cybersecurity Professional Certificate ; an online program on Coursera. This training prepared me for tackling sophisticated security challenges, ensuring robust network and data protection. As part of this certificate, I completed several courses, learned fundamentals, and applied several concepts in cybersecurity, such as managing security risks, protecting data, network security, Python automation, and used tools such as Linux and SQL.

Work Experience

Currently, I work as an AI Performance Tester at (a subsidiary of Surge AI) and Remotasks (a subsidiary of Scale AI), where I work with cutting-edge Generative AI LLMs like Google Gemini, Anthropic Claude 3 Opus, OpenAI GPT-4, and Meta AI, applying Reinforcement Learning from Human Feedback (RLHF) and Human in the Loop (HITL) techniques to optimize model training and evaluation. I assess and rate the outputs (text, audio) of Large Language Models (LLMs) in response to various prompts (text, audio, video), comparing responses from the same or different chatbots to determine which is more accurate or appropriate for a given context. I train LLMs using Natural Language Processing (NLP) techniques for claim verification, model evaluation, model comparison, prompt testing, and data labeling/annotation, focusing on enhancing the performance and accuracy of large language models through RLHF methods and HITL Labeling.

When I'm not coding or working on AI projects, you can find me exploring different LLMs, contributing to open-source projects, and engaging with the vibrant Python community. Enjoy continually learning and staying up to date with the latest trends in the field. In my free time, I enjoy various sports and activities, such as kayaking, playing soccer, snowboarding, ping pong, working out, gaming and reading. I also have a keen interest in cryptocurrency, investing, and financial markets and was actually a Finance Major in my undergrad, but became heavily excited and passionate about the latest technologies and decided to head in that direction.
